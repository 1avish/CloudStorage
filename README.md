# CloudStorage

简易网盘客户端 Android App。

---

## 运行环境依赖

| 依赖 | 版本 |
|------|------|
| Android Studio | Hedgehog (2023.1.1) 或更高 |
| minSdk | 24 (Android 7.0) |
| targetSdk | 34 (Android 14) |
| Kotlin | 1.9+ |
| Compose BOM | 2024.02+ |
| Material3 | 随 Compose BOM |
| Navigation Compose | 2.7+ |
| Lifecycle / ViewModel | 2.7+ |
| Room | 2.6+ |
| Kotlin Coroutines | 1.8+ |

---

## 环境搭建指南

1. 安装 Android Studio Hedgehog 或更高版本。
2. 确保已安装 Android SDK 34 和对应的 Build Tools。
3. 克隆项目后用 Android Studio 打开根目录，等待 Gradle 同步完成。

---

## 项目启动说明

1. 用 Android Studio 打开项目。
2. 连接真机或启动模拟器（API 24+）。
3. 点击 Run（或 Shift + F10）运行 App。
